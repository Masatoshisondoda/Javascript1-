音ゲー

## プロダクトの紹介
・音楽が流れている中でブロックを壊す 
・ただただ高得点を目指す（音が流れるまで時間がかかります）
- 箇条書きにすると
- 読みやすい

## 工夫した点，こだわった点
・音ゲーらしからぬ音ゲー作りを目指した。
- 特に見てほしい点，
- うまくできたと感じている点など．
